# Currency Converter

Автор: Данилова Руслана Андреевна

## Как получить API-ключ

1. Зарегистрируйтесь на https://www.exchangerate-api.com/
2. Создайте аккаунт и получите API-ключ.
3. Вставьте ключ в файл `currency_converter.py` в переменную `API_KEY`.

## Использование

1. Запустите приложение:
2. Введите сумму и выберите валюты.
3. Нажмите "Конвертировать".
4. История отображается ниже.

## Возможные тесты

- Ввод отрицательной суммы -> ошибка.
- Ввод некорректных данных -> ошибка.
- Проверка сохранения истории.
