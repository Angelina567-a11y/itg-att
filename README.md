# Currency Converter
## Автор: Данилова Руслана Андреевна
Это простой графический конвертер валют, созданный на Python с использованием библиотеки Tkinter. Он позволяет:
- Выбирать валюты для конвертации
- Вводить сумму
- Получать результат конвертации по текущим курсам
- Просматривать историю предыдущих конвертаций

## Как работает

При запуске программы автоматически загружаются актуальные курсы валют с использованием бесплатного API. Можно обновлять курсы по кнопке.

После выполнения конвертации результат отображается на экране, а все операции сохраняются в локальной истории, которую можно просматривать в окне.
